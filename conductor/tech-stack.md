# Tech Stack: Jattac.Libs.Web.ResponsiveTable

## Core
- **React:** UI library for component-based development.
- **TypeScript:** Typed JavaScript for robust and maintainable code.

## Build & Bundling
- **Rollup:** JavaScript module bundler for libraries.
- **PostCSS:** Tool for transforming CSS with plugins (using CSS Modules).

## Testing
- **Jest:** Testing framework for JavaScript/TypeScript.
- **React Testing Library:** Testing utilities for React components.

## Linting & Formatting
- **ESLint:** Pluggable linting utility for JavaScript and TypeScript.
- **Prettier:** Opinionated code formatter.
